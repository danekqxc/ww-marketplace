## О проекте

WWM-place — WW(дабл-ю дабл-ю) M(market)place - биржа игровых услуг, маркетплейс игровых ценностей. Предсталвяет из себя онлайн-площадку для торговли товарами, на подобие известных FunPay, Playerok, ggsel. Здесь пользователи могут безопасно продавать и покупать внутриигровую валюту, аккаунты, ключи активации, а также оказывать услуги по бусту или помощи в прохождении внутри различных игровых вселенных.

## 🎮 Функционал

Мы реализовали полноценный цикл работы торговой площадки от регистрации до завершения сделки:

### 1. Витрина и Категории товаров
* Товары разделены по популярным игровым направлениям: **Steam, Minecraft, CS2, Brawl Stars, Roblox** и другие.
* На главной странице работает **живой поиск и фильтрация** — нужный товар или категорию можно найти мгновенно, без постоянных перезагрузок страниц.

### 2. Пользовательский цикл и Экономика
* **Профили «два в одном»:** Каждый зарегистрированный пользователь может быть как покупателем, так и продавцом.
* В личной панели можно загрузить уникальный аватар, посмотреть историю своих заказов и выставленные на продажу товары.
* **Внутренний баланс:** Имитация кошелька. При покупке система проверяет баланс, списывает средства и резервирует их под конкретную сделку.

### 3. Безопасность сделок и Коммуникация
* **Встроенные чаты по заказам:** Как только совершается покупка, для покупателя и продавца создается приватная комната чата, привязанная к этому заказу. Прямо там продавец передает данные (пароли, ссылки, коды доступа), а покупатель их проверяет. В случае, если товар проходит по всем критериям - покупатель может подвердить сделку и деньги уйдут напрямую на счет продавца.
* **Система репутации:** После того как заказ переходит в статус «Выполнен», покупатель может оставить оценку и текстовый отзыв. Это напрямую формирует рейтинг продавца, отсеивая скамеров.

### 4. Избранное (Wishlist)
* Понравившиеся карточки товаров можно в один клик добавить в персональный список желаемого, чтобы не потерять их в общей ленте.

### Установка

1. **Клонируй  репозиторий:**
   ```bash
   git clone [https://github.com/danekqxc/ww-marketplace.git](https://github.com/danekqxc/ww-marketplace.git)
   cd ww-marketplace
   ```

2. **Создай виртуальное окружение:**
   ```bash
   python -m venv .venv
   ```

3. **Активируй виртуальное окружение:**
   * **Для Windows (Command Prompt / PowerShell):**
     ```cmd
     .venv\Scripts\activate
     ```
   * **Для Linux / macOS:**
     ```bash
     source .venv/bin/activate
     ```

4. **Установи необходимые библиотеки:**
   ```bash
   pip install flask werkzeug
   ```

5. **Запусти сервер:**
   ```bash
   python app.py
   ```
